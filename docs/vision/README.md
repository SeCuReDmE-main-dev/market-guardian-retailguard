# V.I.S Guardian Learning Path

V.I.S Guardian means Visual Integrity Sentinel. It is the visual counterpart to V.O.T Guardian.

The purpose is to learn vision as a structured signal system:

- contour: where an object begins and ends;
- motion: how visual state changes over time;
- saliency: what draws visual attention;
- occlusion: what becomes hidden or uncertain;
- track continuity: whether one temporary track remains coherent;
- object presence: whether an object is observed;
- scene contradiction: where visual evidence conflicts with POS, inventory, basket, or cash evidence.

V.I.S Guardian does not create final claims. It creates evidence-bound primitives that Neutro can score.
