# Construction Space Index

Market Guardian / RetailGuard uses two parallel construction spaces before integration:

- Jean-Sébastien Construction Space: human-origin intent, ethics, naming, local reality, and theory.
- Codex Construction Space: AI-origin engineering artifacts, code contracts, tests, and safety checks.
- Integration Space: explicit decisions that record what is merged into the shared tool.

Documents:

- `DUAL_CONSTRUCTION_SPACES.md` explains the two lanes.
- `INTEGRATION_PROTOCOL.md` explains how artifacts merge.

The construction model is implemented in `src/construction/`.
